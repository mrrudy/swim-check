KEY FINDINGS SUMMARY
================================================================================

RECOMMENDED TECH STACK:
  * Monorepo: pnpm workspaces (simple, fast, native)
  * Backend: Express.js + TypeScript
  * Frontend: React 18 + Vite
  * Database: SQLite + better-sqlite3 (sync, simple, fast)
  * Shared Types: @shared/types package (zero runtime cost)
  * API: REST + JSON (simple, standard, cacheable)
  * Scrapers: Strategy pattern + Factory (easy to extend)

CONFIDENCE LEVELS:
  - pnpm workspaces: 95%
  - @shared/types: 90%
  - better-sqlite3: 95%
  - React + Vite: 98%
  - REST API: 97%
  - Strategy pattern scrapers: 99%

UPGRADE PATHS:
  -> pnpm -> Turborepo (if 10+ packages)
  -> @shared/types -> OpenAPI (if 30+ endpoints)
  -> better-sqlite3 -> Drizzle-orm (if 100+ queries)
  -> React -> Next.js (if SSR needed)
  -> REST -> GraphQL (if complex queries)
  -> Strategy -> Plugins (if 50+ scrapers)
